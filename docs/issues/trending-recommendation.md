## Feature: 搜索页增加热门推荐

### 背景
当前搜索页是空白输入框，用户需要知道具体片名才能搜索。对于"不知道看什么"或想浏览热门内容的场景，缺少引导入口。

### 需求
在搜索页增加一个"热门推荐"区域，展示当前流行的影视内容，降低使用门槛。

### 方案建议

#### 数据来源
- **TMDB Trending API**（免费）：`/trending/movie/week`、`/trending/tv/week`
- 当前已配置 TMDB API Key，可直接调用

#### 展示形式
- 搜索框下方展示横向滚动卡片列表
- 每张卡片显示：海报 + 片名 + 年份 + 类型标签
- 点击卡片直接进入搜索/添加该节目

#### 分类切换
- Tab 切换：热门电影 / 热门剧集
- 默认显示本周热门，可扩展为"今日热门"/"本月热门"

### 预期行为
1. 进入搜索页自动加载热门推荐（不依赖用户输入）
2. 点击热门卡片 → 创建追踪并触发首次搜索
3. 加载失败时不阻塞搜索功能，静默降级

### 优先级
中低 — 体验优化，非核心功能。
